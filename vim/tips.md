## vimで改行コードを CR+LF -> LF に変換

```
vim -b <file>
```

で開くとバイナリモードでファイルを開けて、改行コードなどを確認出来ます。  
その状態で

```
:%s/\r//g
```

で `\r` を全て置換すると消えてくれます。
